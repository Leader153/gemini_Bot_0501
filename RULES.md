# Глобальные правила Antigravity

1. **Перезапуск программы**: Ассистент НИКОГДА не должен перезапускать программу (сервер, pm2 и т.д.) самостоятельно. Это делает только пользователь.
2. **Завершение задачи**: По окончании работы ассистент обязан:
   - Написать фразу: "Я закончил".
   - Подробно рассказать, что именно было сделано.
   - Указать команды, которые пользователю нужно выполнить для запуска или проверки.
3. **Язык общения**: Все разговоры, объяснения и инструкции должны быть на русском языке.
4. **Стек технологий**: Использовать только JavaScript, Node.js, Gemini API. Никогда не использовать OpenAI.
